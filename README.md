# shipping-cost-calculator 📦

A phython program that ultilizes package weight in order to calculate and compare different shipping methods to determine the cheapest option. 

## Features
- Calculates Ground Shipping cost based on weight
- Includes Premium Ground Shipping (flat rate)
- Calculates Drone Shipping cost
- Compares all methods to find the cheapest option

## How it works
The program uses conditional statements (if / elif / else) to calculate shipping costs based on weight for each method. It then compares all calculated costs using the min() function to determine the cheapest option.

## What I learned
- How to use conditional logic for multiple scenarios
- How to organize code using separate variables
- How to compare values using Python functions (min)
- How to structure a program with inputs and outputs

## Example Output
Ground Shipping: 1910.0
Ground Shipping Premium: 125
Drone Shipping: 5700.0
Cheapest Option: $ 125
